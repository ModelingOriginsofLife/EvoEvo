• Fitness level

SCORE_SCHEME <string> string( "SUM", "MEAN_MINUS_DEV", "COMBINATORIAL") Set the score scheme used to compute the score from the metabolic concentrations. If SUM, the score is simply the sum of essential metabolite concentrations. If MEAN_MINUS_DEV, the score is the essential metabolite concentrations mean, minus the quadratic error around the mean. If COMBINATORIAL, the score is the combination of all the essential metabolites, such that biggest complexes are built. E.g. if essential metabolites 2, 3, 5 and 7 are expressed with concentrations C3 < C2 < C7 < C5, then the score is s = 1*C3+2*(C2-C3)+3*(C7-C2)+4*(C5-C7). COMBINATORIAL score scheme represents the cell capability to produce replication protein complexes.

ENVIRONMENT_INTERACTION_SCHEME <string> string( "NO_INTERACTION", "INTERACTION") Set the environmental interaction scheme:
  • NO_INTERACTION: The environment does not interact with the population, i.e. cells uptake and release nutrients but do not modify the environment.
  • INTERACTION: The environment interacts with the population, i.e. cells modify the environment by uptaking and releaseing nutrients.
ENVIRONMENT_RENEWAL_SCHEME <string> string( "KEEP_MATTER", "CLEAR_MATTER") Set the environmental renewal scheme:
  • KEEP_MATTER: Previous environmental concentrations are maintained when fresh nutrients are introduced in the environment.
  • CLEAR_MATTER: Previous environmental concentrations are cleared when fresh nutrients are introduced in the environment.
WARNING: CLEAR_MATTER option is activated only if ENVIRONMENT_VARIATION_SCHEME parameter is set to PERIODIC.
ENVIRONMENT_VARIATION_SCHEME <string> string( "RANDOM", "PERIODIC", "CYCLIC") Set the environmental variation scheme:
  • RANDOM: Variation occurs in each grid site with a probability = introduction rate/per timestep
  • PERIODIC: Variation occurs periodically, with a period = 1/(introduction rate)
  • CYCLIC: Variation is cycled, introduced concentrations being weighted by a sinus like function, with a period = 1/(introduction rate)
ENVIRONMENT_VARIATION_LOCALIZATION <string> string( "RANDOM", "GLOBAL", "CENTER") Set the environmental variation scheme:
  • RANDOM: Variation occurs in each grid site with a probability = introduction rate/per timestep
  • GLOBAL: Variation occurs in the whole environment at once
  • CENTER: Variation only occurs at the center of the grid
ENVIRONMENT_INTRODUCTION_RATE <double> double( [0, 1] ) Set the introduction rate of fresh metabolites in the environment
ENVIRONMENT_DIFFUSION_RATE <double> double( [0, 1] ) Set the diffusion rate
ENVIRONMENT_DEGRADATION_RATE <double> double( [0, 1] ) Set the degradation rate
